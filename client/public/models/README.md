Place face-api.js model files in this folder before using face login.

Required models:
- tinyFaceDetector (`tiny_face_detector_model-weights_manifest.json` + shard files)
- faceLandmark68Net (`face_landmark_68_model-weights_manifest.json` + shard files)
- faceRecognitionNet (`face_recognition_model-weights_manifest.json` + shard files)

You can get these files from the official face-api.js repository weights package.
